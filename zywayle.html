// HOW TO CONTRIBUTE
01
Fork the repo

Head to GitHub and fork the zyydata repository to your own account.

02
Add your data

Check the README for format guidelines. Add text, Q&A pairs, or structured entries.

03
Open a PR

Submit a pull request. If it meets quality standards, it gets merged and trained on.

info

Data ownership: All contributions become part of the Zywayle training dataset and are owned by Zywayle, as described in the repository README. Contributing is open to anyone — but ownership of the compiled dataset remains with Zywayle. Read full terms →
